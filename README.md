# AskMyPDF-Intelligent-PDF-Question-Answering-System
AskMyPDF is an AI-powered PDF Question Answering (QA) system that enables users to upload PDF documents and ask natural language questions to retrieve accurate, context-aware answers.
The system leverages Retrieval-Augmented Generation (RAG) using FAISS vector search, HuggingFace embeddings, and Groq-hosted LLMs, all wrapped in an intuitive Streamlit interface.
